## Getting Started

1. **Install dependencies:**
   ```bash
   bun install
   ```

2. **Run locally:**
   ```bash
   bun run dev
   ```

3. **Build for production:**
   ```bash
   bun run build
   ```
   Output will be in `dist/`.

4. **Deploy:**
   - Upload `/dist` to Netlify, Vercel, or static host.
